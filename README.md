# CommercePulse: Predictive Customer Analytics Dashboard

An interactive web-based presentation showcasing a complete data science project focused on predicting customer spending and churn for an e-commerce dataset.


## üöÄ Project Overview

CommercePulse is an interactive web dashboard that walks through a customer analytics project from start to finish. Instead of a static slideshow, this project uses a live web interface to present the data processing steps, model training, and final evaluation. The primary goals of the analysis are:

1.  **Regression:** To predict a customer's total spending (`Monetary` value).
2.  **Classification:** To predict whether a customer is likely to churn.
